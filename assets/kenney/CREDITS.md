## ui-pack/ (UI Pack 2.0)
**Author:** Kenney (www.kenney.nl)
**License:** CC0 — http://creativecommons.org/publicdomain/zero/1.0/
**URL:** https://kenney.nl/assets/ui-pack
**Downloaded from:** https://kenney.nl/media/pages/assets/ui-pack/f651646eab-1718203990/kenney_ui-pack.zip
**Contents:** 870 individual PNG UI elements (buttons, panels, arrows, sliders, checkboxes) in Blue/Green/Grey/Red/Yellow color variants
**Sample sheet:** Sample.png 918x515

### Structure:
```
ui-pack/PNG/<Color>/Default/  — standard size elements
ui-pack/PNG/<Color>/Double/   — 2x size elements
ui-pack/Font/                 — bitmap font
ui-pack/Vector/               — SVG sources
ui-pack/Sounds/               — UI sound effects
```
